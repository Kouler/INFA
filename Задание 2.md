## 1
$$\Large f :\ \ \ (x \equiv (w \lor y)) \lor ((w \rightarrow z) \land (y \rightarrow w))$$

|       |     |       |       | **f** |
| ----- | --- | ----- | ----- | ----- |
| **1** |     |       | **1** | **0** |
|       |     |       | **1** | **0** |
| **1** |     | **1** |       | **0** |
ТАДАМ

| y     | x   | z     | w     | **f** |
| ----- | --- | ----- | ----- | ----- |
| **1** | 0   | 0     | **1** | **0** |
| 0     | 0   | 0     | **1** | **0** |
| **1** | 0   | **1** | 0     | **0** |
Раз глобальный общий знак ИЛИ, то оба подвыражения:
$$\Large(x \equiv (w \lor y)) = 0$$
$$\Large ((w \rightarrow z) \land (y \rightarrow w)) = 0$$

1. $x = 1, \Rightarrow w = y = 0$ ЛИБО $x = 0\ w = 0/1, \ y = 0/1$ , но не оба 0
2. $w = 0 \ \ {and}\ \  z = 1\ \ \ OR \ \ \ y = 0 \ \ {and}\ \  w = 1 \ \ \ OR\ \  BOTH$
