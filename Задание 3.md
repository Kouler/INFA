# Теория / заметки

Когда нужно просуммировать ячейки столбца `A` только если они подходят по условию (к примеру, вторая ячейка `B` текущей записи должна удовлетворять условию), то можно заюзать СУММПРОИЗВ((A1:A99)\*(B1:B99=`значению`)), причём в качестве `значения` **лучше** тоже указать ячейку, в которой лежит это значение.

## 1

Тут в Поступлении суммируем `Количество упаковок` только если соседняя справа ячейка, это <span style="color:#afa">"Поступление"</span>, при этом только 4 магазина из ячеек второго листа <span style="color:rgb(0, 176, 240)">"Магазин"</span>, и только определённый товар (по-аналогии).
 
![[Pasted image 20260115213411.png]]

```excel
=СУММПРОИЗВ((F2:F2273=$F$2)*(E2:E2273)*((C2:C2273=Магазин!A15)+(C2:C2273=Магазин!A12)+(C2:C2273=Магазин!A10)+(C2:C2273=Магазин!A4))*(D2:D2273=Товар!A16))
```
